Project Overview

This project completes the full cycle of "paper reading → code reproduction → WorldArena inference & evaluation → fine-tuning → comparative analysis" for robot video world models. We benchmark ABot-PhysWorld (robot-specific), vanilla Wan2.1 (general video model), and SFT-Wan2.1 (LoRA fine-tuned on RoboTwin) on the WorldArena Challenge Track 1.

Technical Approach

GigaWorld
Large-Scale Video World Model
Video pretraining + embodied finetuning. Public GigaWorld-0-Video-GR1-2b checkpoint run locally as smoke test. Demonstrates the "big video world model" paradigm for robot data generation.
Paper →
ABot-PhysWorld
Robot-Specific World Model
Built on Wan2.1 with 3M robot video SFT + physics-aware DPO. Uses action map / spatial injection for action conditioning. Strong at robot identity and background stability.
Paper →
Vanilla Wan2.1
General Video Generation Baseline
14B diffusion transformer for I2V. Strong visual quality but weak robot identity preservation. Serves as the universal baseline for comparison.
VideoX-Fun + RoboTwin SFT
LoRA Fine-Tuning Pipeline
Our approach: Fine-tune Wan2.1 with LoRA (rank=32) on 2,500 RoboTwin robot manipulation videos using VideoX-Fun framework. Trained to step 2,200/12,500 on 2 GPUs. Checkpoint-2200 used for the 1000-video inference run.
→ Best flow score, improved robot identity
SeedVR Refiner
Post-Processing Enhancement
SeedVR video refiner applied as a post-processing step to improve visual sharpness and reduce artifacts. Dynamic GPU selection for efficient batch processing.

Key Insights

Core observations from comparing robot-specific world models against general video generation models.

01

Robot world models (ABot) are significantly better at preserving robot identity, background consistency, and subject stability compared to generic video models.

02

General video models (Wan2.1) produce superior image quality and aesthetics, but suffer from human hand hallucination and scene drift for robotic domains.

03

Small-scale SFT on robot data (2.5K RoboTwin samples) shows promising improvements in background consistency, subject consistency, and flow score — approaching ABot-level stability.

04

Instruction/action alignment remains the hardest challenge. Even ABot shows only 54% instruction following. Physical plausibility of interactions is still limited.

05

Post-processing (SeedVR) improves visual quality but does not fix underlying action semantics. It's a complementary enhancement, not a substitute for better models.

06

There is a clear gap between "looking good" and "acting correctly" in robot video generation. Future work should focus on action conditioning, DPO-style alignment, and larger robot datasets.
